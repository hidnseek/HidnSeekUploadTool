HidnSeek USBasp support is in public beta stage so please have that in mind, if you encounter any bugs report them on github.

Requirements:
1. USB permissions
2. Install latest WinUSB drivers for your USBasp, you can do so via Zadig (windows users)
3. Select HidnSeek as programmer
4. After plugging in your USBasp, you should see it in the ports area, select it manually if needed

You should be able to flash the firmware,
Enjoy
