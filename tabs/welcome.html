HidnSeek uses a remote server to get latest firmware release, make sure you are connected to the internet.

Quick howto:
1. Stop the tracker. If it's already stopped, go to point 5. Put the tracker flat on table and wait for one red blink
2. Hold the tracker verticaly on the long edge where you can read HidnSeek Logo. Wait for another red blink.
3. Return the tracker and hold it verticaly on the long edge where HidnSeek Logo is reverted. Wait for another red blink.
4. Return the tracker and hold the tracker verticaly on the long edge as (2) where you can read HidnSeek Logo. Wait for another red blink. The tracker must stop with several red blink.
5. Connect the USB cable to the tracker.
3. Connect USB cable to the computer to power it up.
4. Within 10 seconds, click on Flash firmware button
